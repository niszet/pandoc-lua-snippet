# pandoc-lua-snippet README

This is the README for `pandoc-lua-snippet` extenstion. 

## Features

This extension enables you to write Pandoc Lua filter easily. This extenstion includes code snippets for writing Lua filter for Pandoc. Nominal Lua snippets are not included yet. Please use other Lua extension.

## Requirements

None.

## Extension Settings

Set Language to `Lua` to enable this snippets.
